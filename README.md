# Ripple Counter | Verilog HDL

A Verilog HDL implementation of a Ripple Counter demonstrating asynchronous counting using cascaded flip-flops. The project includes synthesizable RTL, simulation, and functional verification.

---

## Features

- Asynchronous Ripple Counter
- Verilog RTL Design
- Synthesizable Implementation
- Functional Testbench
- Xilinx Vivado Compatible

---

## Applications

- Frequency Division
- Digital Counters
- Timing Circuits
- Event Counting

---

## Tech Stack

- Verilog HDL
- Xilinx Vivado
- Digital Electronics
- RTL Design
